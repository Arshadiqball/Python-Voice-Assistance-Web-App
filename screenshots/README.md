# Screenshots

This directory contains demo screenshots of the Voice Assistant Web App.

## Adding a Screenshot

1. Take a screenshot of the running application (browser window showing the interface)
2. Save it as `demo.png` in this directory
3. The README will automatically display it in the Demo section

## Screenshot Tips

- Capture the full browser window showing the interface
- Make sure the sidebar and chat area are visible
- Show a conversation with at least one message exchange
- Recommended size: 1920x1080 or similar aspect ratio
- Format: PNG or JPG
